# Study_Planner_Project
Study Planner is a final year project developed during my B.Sc. IT course. This project helps students organize their study schedules, manage tasks, and plan their learning activities efficiently.

# 3.Forgot Password  Module
<img width="663" height="426" alt="image" src="https://github.com/user-attachments/assets/08bf0c36-bad4-4af4-adcd-d50ee521efd1" />

### Description : 
The Forgot Password screen is a security recovery module that allows users to regain access to their account if they forget their password. It uses a two-step process: first, the user enters their registered email to Send OTP (One-Time Password); second, they must Verify OTP to proceed with setting a new password. A "Back to Login" link is also provided. 


# 4.Reset Password  Module 
<img width="663" height="485" alt="image" src="https://github.com/user-attachments/assets/1e42e1e5-9973-44ad-9f43-cdd737b32601" />

### Description : 
The Reset Password screen is the final step in the password recovery process. It requires the user to re-enter the OTP (if needed) and then set and confirm a New Password to finalize account access restoration.             


# 5.Dashboard Page 
<img width="663" height="387" alt="image" src="https://github.com/user-attachments/assets/84b92ed9-c134-4549-812d-5e900a2ac652" />

### Description: 
The Dashboard acts as the user's academic control center. It welcomes the user and provides a snapshot of their progress, summarizing Total Classes, Assignments, Study Hours, and Completion Rate. The screen features a crucial sidebar for quick navigation to Activity, Class Schedule, Assignments, and Study Sessions, and highlights the most immediate deadline under Upcoming Assignments.

# 6.Activity Page 
<img width="1036" height="595" alt="image" src="https://github.com/user-attachments/assets/94898e12-ee2a-4127-bb1c-281326970a28" />

### Description: 
The Activity Screen allows the user to Track their Activity and progress. It displays visual metrics for the Weekly Study Goal and Assignment Completion (e.g., 2 of 4 completed). It also provides a history of Recent Study Sessions, showing the subject, date, and duration (e.g., Science, 130 min).

# 7.Class Schedule Page 
<img width="1036" height="613" alt="image" src="https://github.com/user-attachments/assets/6f6c8c04-7210-4aeb-8da6-6bd54ea1fc84" />

### Description: 
The Class Schedule Screen is designed to help the user manage their daily and weekly classes. It displays a list of courses (e.g., Java Programming, DBMS, Linux Practical), including the Instructor's Note, Time, and Day. Users can Add, Edit, or Delete classes directly from this view.

# 8.Assignment  Page 
<img width="1036" height="653" alt="image" src="https://github.com/user-attachments/assets/0445ae22-3ba4-4e7e-9c17-eb3b044f143a" />

### Description: 
The Assignments Screen allows the user to track and manage all their assignments. It lists each assignment with its Title, Description, Due Date, and Status (Pending/Completed). Users can filter the list, add New Assignments, and use the Edit or Delete actions for existing entries.

# 9.Study Sessions  Page 
<img width="663" height="391" alt="image" src="https://github.com/user-attachments/assets/9f28202c-a720-43a9-b9a2-98e81b267e47" />

### Description: 
The Study Sessions Screen is used to monitor daily and weekly study progress and track time logs. It displays metrics for Today's Study Time and This Week's total. Users can Add a Session and view a table of Recent Sessions, which details the Subject, Duration, Date, and Notes, with options to Edit or Delete past entries.

# 10.Logout  Page 
<img width="663" height="385" alt="image" src="https://github.com/user-attachments/assets/8442804c-2b81-486a-b012-d7e2333d03ba" />

### Description: 
The Confirm Logout screen is a security confirmation modal that appears when the user attempts to log out. It asks the user to confirm their action with the message, "Are you sure you want to logout?", providing two clear options: "Yes, Logout" to exit the application, or "Cancel" to remain on the current screen.
